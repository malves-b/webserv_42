void ServerSocket::startSocket ( std::string const &  port)

Creates, binds, and configures a non-blocking server socket.

Uses getaddrinfo() to support both IPv4 and IPv6 and binds to the specified port. The socket is configured with SO_REUSEADDR and set to non-blocking mode.

Parameters
portPort number to bind the socket to (as string).
Exceptions
