HttpResponse &  res 
)
static

Generates and serves an HTML directory listing when autoindex is enabled.

Reads the contents of the resolved directory, formats each entry into a table row, and injects the result into the HTML template. Adds delete buttons for files and ensures template placeholders are replaced before sending the response.
